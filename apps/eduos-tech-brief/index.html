🤖 Agentic AI — القلب الذكي

EduOS هو أول منصة تعليمية خليجية تعتمد Agentic AI بالكامل. كل وظيفة في المنصة لها مستوى استقلالية من A إلى D. Edge Function مركزية واحدة — agent_brain — تخدم جميع الوظائف الـ 26.

المستوى المعنى مثال
D — مستقل كاملينفذ تلقائياً دون إشعارتنظيف بيانات المعلمين البدلاء
C — ينفذ + يُشعرينفذ ثم يُرسل إشعاراًتقرير الوالدين الأسبوعي
B — يطلب موافقةيقترح وينتظر الموافقةتعديل درجة طالب/ة
A — يُبلِّغ فقطيرصد ويُعلم دون تدخلرصد النمط السلوكي

🛡️ الأمان والامتثال

🔐 طبقة المصادقة

  • JWT Supabase مع دور مُضمَّن
  • كل طلب يحمل دوره الصريح
  • لا OR_KEY في أي صفحة إطلاقاً
  • sessionStorage فقط — لا localStorage

📋 RLS + Edge Functions

  • Row Level Security على كل جدول
  • anon key = قراءة فقط
  • كل الكتابات عبر Edge Functions
  • Gemini API خلف askAI edge-function

🔍 EduOS Shield

  • مراقبة JS + زمن الاستجابة
  • كشف الروابط المكسورة
  • إبلاغ عبر bug_reports
  • تنبيهات فورية للمشرف/ة

⏱️ الجلسة والخصوصية

  • تسجيل خروج تلقائي بعد 15 دقيقة
  • حفظ التبويب عبر URL hash
  • لا بيانات طالب/ة في localStorage
  • بيانات الأطفال محمية بـ RLS صارم
