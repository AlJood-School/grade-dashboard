الجود — EduOS

البوابة التعليمية الذكية الأم. نظام إدارة مدرسي متكامل بذكاء اصطناعي حقيقي. يغطي 7 أدوار ويدعم KG حتى الثانوية.

🧠

نفس

منصة الصحة النفسية. تتابع الحالة النفسية للطالب/ة بصمت وتنبّه المختصين عند الحاجة دون كشف هوية الطالب/ة.

✏️

مداد

منصة الكتابة الإبداعية المدعومة بالذكاء الاصطناعي. تنمّي مهارات الكتابة والتعبير عند الطلاب/ات بطريقة تفاعلية.

🔬

عمق

منصة التعلم العميق والبحث العلمي. تدعم التفكير النقدي والاستقصاء العلمي للمراحل المتقدمة.

🎮

أثير — Atheer

المرافق التعليمي الذكي للأطفال. يستخدم Phaser.js + ElevenLabs + Claude API لتجربة تعلّم تفاعلية وآمنة.
