Basic Level المستوى الأساسي
🎯 Your Task: Make the micro:bit show YOUR robot's name when you press Button A!
المهمة: اجعل المايكروبت يعرض اسم روبوتك عند ضغط الزر A
📦 Your Code Blocks
👆 on button A pressed EVENT
💬 show string ??? ACTION

🔧 How To Do It

1 Go to Input → drag "on button A pressed"
2 Go to Basic → drag "show string" inside
3 Type your robot's name in the white box
4 Click Download and test! 🎉
📗
Intermediate المستوى المتوسط
🎯 Your Task: Press A → show name. Press B → show a heart ❤️
المهمة: الزر A = الاسم، الزر B = قلب
📦 Your Code Blocks
👆 on button A pressed EVENT
💬 show string "Hi" ACTION
👆 on button B pressed EVENT
❤️ show icon ♥ ACTION

🔧 How To Do It

1 Input → "on button A pressed"
2 Basic → "show string" → type "Hi"
3 Input → "on button B pressed"
4 Basic → "show icon" → pick ❤️ heart
5 Download and test both buttons! 🥳
⭐
Advanced المستوى المتقدم
🎯 Your Task: A → Hello · B → ❤️ · A+B → Red lights · Loop → Robot name!
المهمة: ٤ بلوكات كاملة — روبوت ذكي يتحرك دائماً
📦 Your Code Blocks
👆 on button A pressed EVENT
💬 show string "Hello" ACTION
👆 on button B pressed EVENT
❤️ show icon ♥ ACTION
✌️ on button A+B pressed EVENT
🔴 set neopixels to RED ACTION
🔁 forever (loop) LOOP
💬 show string "MY ROBOT" ACTION

🔧 How To Do It

1 Add 3 × Input blocks (A, B, A+B)
2 Add Basic → "forever" loop block
3 Add Neopixel extension for colored lights
4 Test all 4 buttons + loop! 🏆
